# Plans Folder

One folder for everything: training plans, session logs, brainstorm notes.

## Structure

| File | Purpose |
|------|---------|
| `overview.md` | Research goal, model architecture, key concepts — doesn't change often |
| `brainstorm.md` | Running ideas, things to try, open questions |
| `session_01_baseline.md` | Sessions 1 & 2 — original runs, what failed and why |
| `session_02_prep.md` | Session 3 (next run) — all changes made, goals, what to watch |

## Workflow after each training session

1. Copy `session_02_prep.md` → `session_03_prep.md` (increment number)
2. Fill in results in the completed session file (BLEU, loss curve, sample outputs)
3. Update `brainstorm.md` with new ideas based on what you observed
4. Update the new session prep file with what you plan to change next
